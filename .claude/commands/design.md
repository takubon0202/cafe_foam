# デザインエージェント実行

`agents/02_design.md` を読み込み、以下を実行します：

1. デザイン方向性の決定
2. カラーパレット選定
3. タイポグラフィ設計
4. コンポーネント仕様作成

## 禁止事項の確認
- 紫グラデーション NG
- 絵文字多用 NG
- Inter/Roboto/Arial NG
- 汎用的なAIデザイン NG

## 推奨アプローチ
- カフェらしい温かみ
- 和モダンの繊細さ
- 印刷したくなる美しさ
- 情報の優先度が明確な階層構造

## 引数
- $TASK: デザインタスクの説明

$TASK に基づいてデザインを作成してください。
