# 開発フルサイクル実行

以下の順序で全サブエージェントを実行し、品質を担保します。

## 実行手順

1. **ペルソナ確認**
   - `agents/01_persona.md` を読み込み
   - ターゲットユーザー要件を確認

2. **デザイン設計**
   - `agents/02_design.md` を読み込み
   - デザイントークン、コンポーネント仕様を作成
   - AIっぽいデザインを徹底回避

3. **コーディング**
   - `agents/03_coding.md` を読み込み
   - HTML/CSS/JSを実装
   - アクセシビリティ、パフォーマンスを考慮

4. **画像生成**（必要時）
   - `agents/04_image.md` を読み込み
   - 必要な画像を生成・最適化

5. **バックエンド**（必要時）
   - `agents/05_backend.md` を読み込み
   - データ構造、ストレージを実装

6. **デバッグ**
   - `agents/06_debug.md` を読み込み
   - テスト実行、バグ修正

7. **評価**
   - `agents/07_evaluator.md` を読み込み
   - 品質評価、合格判定
   - 不合格の場合は該当エージェントに差し戻し

## 引数
- $TASK: 開発タスクの説明

## 出力
- 実装コード
- テスト結果
- 評価レポート
- 合格/不合格判定

$TASK の内容に基づいて開発を進めてください。
必ず80点以上で合格するまでイテレーションしてください。
