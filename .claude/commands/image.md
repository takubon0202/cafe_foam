# 画像生成エージェント実行

`agents/04_image.md` を読み込み、以下を実行します：

1. 画像要件の確認
2. 生成プロンプト作成
3. 画像最適化指示
4. alt属性テキスト作成

## スタイルガイドライン
- ナチュラル、温かみのあるスタイル
- AIっぽい過度な完璧さは避ける
- カフェらしい素材感

## 画像カテゴリー

### メニュー写真
- サイズ: 400x400px (1:1)
- フォーマット: WebP + JPGフォールバック
- スタイル: 真上または45度俯瞰、白いカップ、木のテーブル

### 手順写真
- サイズ: 800x600px (4:3)
- フォーマット: WebP + JPGフォールバック
- スタイル: 手元クローズアップ、明確なアクション

### アイコン
- サイズ: 48x48px, 96x96px
- フォーマット: SVG推奨
- スタイル: 線画、2色以内

## 禁止事項
- AIっぽい過度に完璧な画像
- 不自然なツヤや光沢
- ストックフォト的な構図
- 3Dレンダリング風

## 引数
- $TASK: 画像生成タスクの説明

$TASK に基づいて画像仕様を作成してください。
