# /inventory - 在庫管理コマンド

在庫管理システムの構築・更新を行います。

## 実行内容

1. **GASスクリプトの確認・更新**
   - スプレッドシートからデータを取得するAPIの実装
   - CORS対応、エラーハンドリング

2. **フロントエンドの更新**
   - 在庫一覧の表示
   - 発注アラートの視覚化
   - 保管場所情報の表示

3. **スタイルの調整**
   - カフェらしいデザイン
   - 発注が必要なアイテムのハイライト

## 使用エージェント

- `08_inventory.md` - 在庫管理エージェント

## 関連スキル

- `inventory-management.skill.md` - 在庫管理スキル

## パラメータ

- `$ARGUMENTS` - オプション引数（例: `update`, `check`）

## 実行例

```
/inventory          # 在庫管理システム全体を確認・更新
/inventory update   # GAS APIとフロントエンドを更新
/inventory check    # 現在の在庫状況をチェック
```

## 対象ファイル

- `index.html` - 在庫管理セクション
- `script.js` - 在庫取得・表示ロジック
- `data.js` - 在庫データ構造
- `styles.css` - 在庫管理スタイル
- `gas/inventory-api.js` - GASスクリプト

## 注意事項

- スプレッドシートは読み取り専用
- GASのデプロイが必要
- スプレッドシートIDは機密情報として扱う
