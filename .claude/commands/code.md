# コーディングエージェント実行

`agents/03_coding.md` を読み込み、以下を実行します：

1. セマンティックHTML実装
2. CSS設計（BEM + カスタムプロパティ）
3. JavaScript実装
4. アクセシビリティ対応
5. 印刷スタイル対応

## コーディング規約
- セマンティックなHTML構造を徹底
- BEM命名規則 + CSS変数
- バニラJS（フレームワーク不使用）
- WCAG 2.1 AA準拠

## パフォーマンス基準
- First Contentful Paint: < 1.5s
- Largest Contentful Paint: < 2.5s
- Cumulative Layout Shift: < 0.1

## 引数
- $TASK: コーディングタスクの説明

$TASK に基づいて実装してください。
