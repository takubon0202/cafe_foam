# 品質チェックスキル

## 概要
Cafe Foamの成果物品質を評価するための基準と手法

## 評価カテゴリー

### 1. デザイン品質 (25点満点)

#### AIっぽさ回避 (8点)
```yaml
8点: 完全に回避、手作り感あり
6点: ほぼ回避、わずかに気になる点
4点: 部分的にAIっぽい要素あり
2点: 明らかにAIデザイン
0点: 完全にAI生成風
```

チェック項目:
- [ ] 紫グラデーションなし
- [ ] 絵文字装飾なし
- [ ] Inter/Roboto/Arial なし
- [ ] 均一なカードグリッドなし
- [ ] 白背景+青アクセントなし

#### カフェらしさ (6点)
```yaml
6点: カフェの雰囲気を完璧に再現
4点: カフェ感があるが改善余地あり
2点: カフェ感が薄い
0点: カフェとは無関係
```

#### 一貫性 (5点)
```yaml
5点: すべての要素が統一
4点: ほぼ統一
3点: 部分的に不統一
2点: 統一感が薄い
1点: バラバラ
```

#### 独自性 (6点)
```yaml
6点: 記憶に残る独自のデザイン
4点: ある程度の独自性
2点: よくあるデザイン
0点: 完全に汎用的
```

### 2. ユーザビリティ (25点満点)

#### 情報発見性 (8点)
```yaml
テスト方法: 「カフェラテの作り方を見つける」タスク
8点: 2タップ以内
6点: 3タップ
4点: 4タップ
2点: 5タップ以上
0点: 見つからない
```

#### モバイル操作性 (8点)
```yaml
8点: 片手で完全操作可能
6点: ほぼ片手で可能
4点: 時々両手が必要
2点: 両手必須
0点: 操作困難
```

#### 緊急時対応 (5点)
```yaml
5点: 混雑時でも問題なく使える
3点: 若干のストレス
1点: 使いづらい
```

#### 印刷対応 (4点)
```yaml
4点: 印刷完璧
3点: 軽微な問題
2点: 読みづらい
1点: 印刷不可
```

### 3. 技術品質 (25点満点)

#### Lighthouse スコア (10点)
```yaml
10点: 全項目95点以上
8点: 全項目90点以上
6点: 全項目80点以上
4点: 全項目70点以上
2点: 70点未満あり
```

#### アクセシビリティ (8点)
```yaml
8点: WCAG 2.1 AAA準拠
6点: WCAG 2.1 AA準拠
4点: 主要項目準拠
2点: 基本的な対応のみ
0点: 未対応
```

#### コード品質 (7点)
```yaml
7点: 模範的なコード
5点: 良好
3点: 普通
1点: 改善必要
```

### 4. 事業適合性 (25点満点)

#### ペルソナ適合 (10点)
```yaml
田中さんテスト:
- 5点: 接客中に問題なく使える
- 3点: 若干の問題あり
- 1点: 使用困難

佐藤さんテスト:
- 5点: 新人教育に最適
- 3点: 使用可能
- 1点: 教育に不適
```

#### 運用効率 (8点)
```yaml
8点: 更新・管理が容易
6点: 若干の手間
4点: かなりの手間
2点: 更新困難
```

#### ブランド表現 (7点)
```yaml
7点: Cafe Foamらしさ完璧
5点: ブランドを感じる
3点: 普通のカフェサイト
1点: ブランド未表現
```

## 合格基準
```yaml
合格: 80点以上
条件付き合格: 70-79点
不合格: 70点未満
```
