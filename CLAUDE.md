# Cafe Foam - Claude Code プロジェクト設定

## プロジェクト概要
- **名称**: Cafe Foam 運営マニュアル
- **目的**: デロンギ マグニフィカS を使用したカフェの運営マニュアルサイト
- **ターゲット**: カフェスタッフ（新人バリスタ）、店舗責任者
- **技術スタック**: HTML5, CSS3, JavaScript (Vanilla)

## デザイン原則

### 絶対禁止事項（AIっぽいデザイン回避）
```
❌ 紫色のグラデーション背景
❌ 絵文字の装飾的多用
❌ Inter, Roboto, Arial などの汎用フォント
❌ 角丸カード + 薄い影の定型パターン
❌ 白背景 + 青アクセントの組み合わせ
❌ 過度にクリーンで無個性なデザイン
❌ 汎用的なアイコンセット
❌ ストックフォト的な画像
```

### 推奨デザイン方向性
```
✓ カフェらしい温かみ: アースカラー、木目調、クラフト感
✓ 実用性重視: マニュアルとして即座に情報を見つけられる構造
✓ 印刷対応: プリントアウトしても見やすいレイアウト
✓ 和モダン: 抹茶ラテ、ほうじ茶ラテを扱うため和の要素
✓ 素材感: 紙、リネン、木などのテクスチャ
✓ 手作り感: 手書き風のアクセント要素
```


### 画像生成フロー
```
1. 用途・サイズ・縦横比を決定
2. Pythonスクリプトでプロンプトを作成（generate_ng_images.py を参考）
3. Gemini API で画像生成
4. images/ フォルダに保存
5. data.js に画像パスとalt属性を追加
6. script.js で画像表示ロジックを実装
7. styles.css で画像カードスタイルを追加
```

### 画像サイズ規定
| カテゴリ | サイズ | 縦横比 |
|----------|--------|--------|
| メニュー写真 | 400x400px | 1:1 |
| 手順写真 | 800x600px | 4:3 |
| イラスト | 800x600px | 4:3 |
| バナー画像 | 1200x400px | 3:1 |

## サブエージェント連携

開発時は以下のスラッシュコマンドで各エージェントを実行：

| コマンド | 役割 | 説明 |
|----------|------|------|
| `/develop` | フルサイクル | 全エージェント順次実行、80点合格まで |
| `/design` | デザイン | ビジュアル設計 + 画像生成 |
| `/code` | コーディング | HTML/CSS/JS実装 |
| `/image` | 画像生成 | Gemini API画像生成・最適化 |
| `/debug` | デバッグ | テスト・バグ修正 |
| `/evaluate` | 評価 | 品質評価・合格判定 |

## ターゲットペルソナ

### プライマリ: 新人バリスタ「田中さん」（22歳）
- 勤務1ヶ月目、スマホ操作は得意
- 接客中に片手でレシピを素早く確認したい
- 緊急時にパニックにならないUI必要

### セカンダリ: 店長「佐藤さん」（35歳）
- 運営5年、新人教育担当
- マニュアル更新・共有を効率化したい
- 衛生記録の管理が必要

## 判断基準

すべての実装は以下の質問で検証：
1. 「田中さんが接客中に片手でスマホを見ながら操作できるか？」
2. 「佐藤さんがスタッフに説明なしで使わせられるか？」
3. 「緊急時（混雑、トラブル）でも使いやすいか？」
